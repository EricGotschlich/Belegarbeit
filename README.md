# Logistik E‑Learning – Belegarbeit (HTW)

## Autor

Eric Gotschlich
Matrikelnummer: 593929

## Projektbeschreibung

https://belegarbeit-216b.onrender.com/

Diese Anwendung ist eine interaktive E‑Learning‑Komponente für das Fach Produktionswirtschaft/Logistik an der HTW.
Sie visualisiert **3 Methoden** (ABC‑/XYZ‑Analyse, Stücklistenanalyse, Bestellverfahren) sowie **einen logistischen Zielkonflikt** (Bestände ↔ Lieferfähigkeit).

## Technologien

- **Vue.js 3**
- **Vite**
- **Bootstrap 5**
- **Material Design Icons**

## Seitenübersicht

- **Home**: Szenario/Unternehmensvorstellung + roter Faden
- **ABC/XYZ‑Analyse**: Verbrauchswert, Wertanteil (kumuliert), Variationskoeffizient + Quiz
- **Stücklistenanalyse**: Mengenstückliste/Baukastenstückliste/Strukturstückliste/Gozinto‑Graph + Übung
- **Bestellverfahren (Vorratsbeschaffung)**: Bereitstellverfahren/Meldebestandsverfahren/Zykluszeitverfahren + Meldebestand‑Rechner
- **Zielkonflikt**: Bestände ↔ Lieferfähigkeit (Regler + Quiz)
